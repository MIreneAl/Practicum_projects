# Окружение, использованное при тестировании API Яндекс.Прилавка

## Клиентская часть
- **Операционная система:** Windows 10 (64-bit)
- **Браузер:** Opera GX версия 116.0.5366.157
- **Инструмент для API-запросов:** Postman (версия для Windows)
- **Доступ к серверу:** SSH-клиент (например, PuTTY или встроенный в ОС)

## Серверная часть
- Удалённый стенд Яндекс.Прилавка (доступ по SSH при необходимости)

## Для воспроизведения дефектов или повторного тестирования потребуется
- Любая ОС семейства Windows (рекомендуется Windows 10/11)
- Установленный Postman (импортировать коллекции по ссылкам в README)
- Доступ к стенду (учётные данные предоставляются отдельно)
