# Практикум: проекты по тестированию

## Список проектов

### [Тестирование веб-приложения «Помоги лично»](./pomogi-lichno-testing)
Благотворительный фонд. Командное исследовательское тестирование с фокусом на функциональность, UI/UX, API, адаптивность и кроссплатформенную совместимость. Анализ требований и макетов (Figma), разработка тестовой документации, тестирование основных модулей сайта с применением техник тест-дизайна. Ведение и приоритизация дефектов в баг-трекере.

### [Функциональное тестирование веб-приложения «Яндекс.Самокат»](./Scooter_Order_Testing)
Полный цикл тестирования: от анализа требований до заключения о готовности к релизу. Анализ требований, разработка тестовой документации, выявление неточностей и «серых зон». Проверка вёрстки формы «Для кого самокат», логики обеих форм, шапки сайта, работы с куки.

### [Тестирование API бэкенда для сервиса доставки продуктов «Яндекс.Прилавок»](./Yandex_Prilavok_API_Testing)
Комплексное тестирование API новой версии приложения (API v3.1.1). Анализ спецификации API (Apidoc) и требований, выявление «серых зон» для формирования тестового покрытия. Разработка тестовой документации, тестирование эндпоинтов расчёта доставки и работы с корзиной.
